## 📌 业务目标

个股研报菜单用于展示券商和机构发布的个股研究报告，支持股票检索、评级变动、目标价、AI 摘要和自选股票研报提醒。

需要展示：

- 报告标题
- 股票代码
- 股票名称
- 机构名称
- 分析师
- 报告日期
- 评级
- 评级变动
- 目标价
- 摘要
- 原文链接
- 行业
- 关键词
- 是否自选相关
- 是否持仓相关
- AI 摘要

---

## 👤 主要参与角色

| 角色 | 类型 | 权限描述 |
| --- | --- | --- |
| 股票用户 | 人类用户 | 检索和阅读个股研报摘要 |
| 自选用户 | 人类用户 | 查看自选股票相关研报提醒 |
| 研报采集系统 | 系统 | 采集研报元数据和摘要 |
| AI系统 | 系统 | 生成研报摘要、观点和风险提取 |
| 管理员 | 人类用户 | 配置研报来源和采集频率 |

---

## 🧩 业务流程图（文字描述）

1. 研报采集系统同步机构研报元数据
2. 系统解析股票、机构、评级、目标价、摘要和链接
3. 系统关联股票、行业和用户自选
4. 用户进入【个股研报】菜单
5. 用户按股票、机构、评级、日期、关键词筛选
6. 用户点击研报查看摘要和原文链接
7. 用户点击 AI 摘要提取投资逻辑、风险和关键假设
8. 自选/持仓股票有新研报时可触发提醒

## 📊 页面字段说明

| 字段 | 中文名 | 展示要求 | 筛选/排序 |
| --- | --- | --- | --- |
| report_id | 研报ID | 系统内部 ID | 不展示 |
| title | 报告标题 | 可点击 | 支持搜索 |
| code | 股票代码 | 可点击 | 支持搜索 |
| name | 股票名称 | 展示标签 | 支持搜索 |
| institution | 机构 | 机构名称 | 支持筛选 |
| analyst | 分析师 | 多人逗号分隔 | 支持筛选 |
| report_date | 报告日期 | YYYY-MM-DD | 支持筛选 |
| rating | 评级 | 买入/增持/中性等 | 支持筛选 |
| rating_change | 评级变动 | 上调/下调/维持 | 支持筛选 |
| target_price | 目标价 | 空值显示 `--` | 支持排序 |
| summary | 摘要 | 可展开 | 支持搜索 |
| source_url | 原文链接 | 新窗口打开 | 支持查看 |
| is_watchlisted | 是否自选相关 | 本人私有状态 | 支持筛选 |

## 🧪 流程测试要点

- ✅ 可查看摘要和原文链接
- ✅ 可生成 AI 研报摘要

- ✅ 研报去重有效
- ✅ 原文链接可打开或提示失效

- ✅ 原文链接失效不影响摘要展示

## 🛡️ 安全与合规

### 数据安全
- 🔒 研报全文版权内容不全文落库
- 🔒 保存摘要、元数据和来源链接
- 🔒 自选/持仓相关状态仅本人可见

### 访问控制
- 🔐 普通用户查看公开研报摘要
- 🔐 管理员配置研报来源

### 合规要求
- 📋 展示研报来源、机构和日期
- 📋 AI 摘要不替代原研报观点
- 📋 研报内容不构成平台投资建议

### 日志审计
- 📊 记录研报采集任务
- 📊 记录 AI 摘要请求
- 📊 记录用户打开原文链接
